Thanks for deploying the Discovery Platform stack.

To inspect the rollout status:
  kubectl --namespace {{ default .Release.Namespace .Values.namespace.name }} get pods -l app.kubernetes.io/instance={{ .Release.Name }}

Backend ingress endpoint:
  kubectl --namespace {{ default .Release.Namespace .Values.namespace.name }} get ingress {{ printf "%s-backend" (include "discovery-platform.fullname" .) }}

Frontend ingress endpoint:
  kubectl --namespace {{ default .Release.Namespace .Values.namespace.name }} get ingress {{ printf "%s-frontend" (include "discovery-platform.fullname" .) }}

Port-forward backend locally:
  kubectl --namespace {{ default .Release.Namespace .Values.namespace.name }} port-forward svc/{{ printf "%s-backend" (include "discovery-platform.fullname" .) }} 8080:{{ .Values.backend.service.port }}
